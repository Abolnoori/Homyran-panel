# راهنمای نصب پنل مدیریت املاک هومیران

## مراحل نصب

### 1. نصب دیتابیس

ابتدا دیتابیس MySQL را ایجاد کنید:

```sql
CREATE DATABASE homyran_panel CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

سپس فایل SQL را اجرا کنید:

**روش 1: از طریق phpMyAdmin**
- وارد phpMyAdmin شوید
- دیتابیس `homyran_panel` را انتخاب کنید
- به تب Import بروید
- فایل `database/schema.sql` را انتخاب و Import کنید

**روش 2: از طریق خط فرمان**
```bash
mysql -u root -p homyran_panel < database/schema.sql
```

### 2. تنظیمات دیتابیس

فایل `config/database.php` را باز کرده و اطلاعات دیتابیس خود را وارد کنید:

```php
define('DB_HOST', 'localhost');    // آدرس سرور دیتابیس
define('DB_USER', 'root');         // نام کاربری دیتابیس
define('DB_PASS', '');             // رمز عبور دیتابیس
define('DB_NAME', 'homyran_panel'); // نام دیتابیس
```

### 3. قرار دادن لوگو (اختیاری)

لوگوی خود را در پوشه `assets/` با نام `logo.svg` یا `logo.png` قرار دهید.

### 4. راه‌اندازی سرور

**روش 1: استفاده از PHP Built-in Server**
```bash
php -S localhost:8000
```

**روش 2: استفاده از XAMPP/WAMP**
- فایل‌های پروژه را در پوشه `htdocs` یا `www` قرار دهید
- Apache و MySQL را راه‌اندازی کنید
- به آدرس `http://localhost/Homyran-panel` بروید

**روش 3: استفاده از Nginx**
- تنظیمات Virtual Host را برای پروژه انجام دهید
- Document Root را به پوشه پروژه تنظیم کنید

### 5. ورود به سیستم

پس از راه‌اندازی، به آدرس زیر بروید:
- `http://localhost:8000/login.php` (برای PHP Built-in Server)
- `http://localhost/Homyran-panel/login.php` (برای XAMPP/WAMP)

**اطلاعات ورود پیش‌فرض:**
- نام کاربری: `admin`
- رمز عبور: `admin123`

⚠️ **توجه:** پس از اولین ورود، حتماً رمز عبور را تغییر دهید!

## عیب‌یابی

### مشکل: خطا در اتصال به دیتابیس
- بررسی کنید که MySQL در حال اجرا باشد
- اطلاعات دیتابیس در `config/database.php` را بررسی کنید
- اطمینان حاصل کنید که دیتابیس ایجاد شده است

### مشکل: صفحه سفید نمایش داده می‌شود
- خطاهای PHP را در فایل لاگ بررسی کنید
- مطمئن شوید که PHP 7.4 یا بالاتر نصب است
- بررسی کنید که تمام فایل‌ها به درستی آپلود شده‌اند

### مشکل: لوگو نمایش داده نمی‌شود
- بررسی کنید که فایل لوگو در پوشه `assets/` وجود دارد
- نام فایل باید `logo.svg` یا `logo.png` باشد
- دسترسی فایل را بررسی کنید

## پشتیبانی

در صورت بروز مشکل، لطفاً با تیم توسعه تماس بگیرید.
